BLINK

AN EVENT SOLUTION APPLICATION UNLIKE ANYTHING YOU'VE SEEN BEFORE

This project was made for 50.001 Introduction to Information System & Programming course together with 5 more members.

About the Project:
BLink is an event solution application that in brief utilises facial recognition technology and simple user interface to engage customers in a new and meaningful way.

Besides using facial recognition for attendance checking at the event, we also utilise it to connect people at the event together. This is done through users taking selfies together and the back end server will recognise the users in the image to link them together.
