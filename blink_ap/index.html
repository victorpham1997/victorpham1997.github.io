BLINK

AN EVENT SOLUTION APPLICATION UNLIKE ANYTHING YOU'VE SEEN BEFORE
Java Android Python Computer Vision Firebase


About the Project:
BLink is an event solution application that in brief utilises facial recognition technology and simple user interface to engage customers in a new and meaningful way.

The application has 2 featues:

  • It acts as a platform that allows event organisers to advertise and post their events, and the users can browse through avaiable event and sign up for them through our app.
  • Our application also use facial recognition for attendance checking at the event. We also use facial recognition to conenct people at the event together. This is done by users taking picture together and our application will automatically recognise and link the photo together.

    • This project was made for 50.001 Introduction to Information Systems & Programming course together with 5 more members. The project won 3rd place for the Mitsubishi Corporation Introduction to Information Systems & Programming Award
