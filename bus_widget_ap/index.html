About the Project:


Singapore Bus Timing Widget is an application that allows you to look up and save the next arrival timings of Singapore buses at specific bus stops. This information can be displayed on your home page through multiple widget styles.

This project is developed out of the inconveience I experience daily, so I want to create a widget app which can tell me when my frequently used bus is arriving.
